## Юридические документы и Онбординг

Все участники проекта (разработчики, администраторы, Staff) обязаны пройти регистрацию:

1. Ознакомьтесь с [Соглашением об участии (Participant Agreement)](PARTICIPANT_AGREEMENT.md) и [Кодексом поведения](CODE_OF_CONDUCT.md).
2. Заполните [Яндекс.Форму для онбординга](https://forms.yandex.ru/u/YOUR_FORM_ID/).
3. Подробная структура анкеты описана в [ONBOARDING_FORM_STRUCTURE.md](ONBOARDING_FORM_STRUCTURE.md).
