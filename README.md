# MinePrime Organization Profile (.github)

Этот репозиторий содержит глобальные настройки и шаблоны для организации **MinePrime-Ru**.

## Содержимое

- `CONTRIBUTING.md` — Общие правила внесения вклада.
- `CODE_OF_CONDUCT.md` — Кодекс поведения.
- `.github/PULL_REQUEST_TEMPLATE.md` — Шаблон Pull Request по умолчанию.
- `.github/CODEOWNERS` — Правила владения кодом по умолчанию.

## Как это работает

GitHub автоматически использует файлы из этого репозитория для всех остальных репозиториев организации, если в них отсутствуют собственные аналогичные файлы.

Юридические документы проекта (Agreement, Acceptance Template) находятся в основном репозитории управления: [mineprime-governance](https://github.com/MinePrime-Ru/mineprime-governance).

# MinePrime Organization Profile (.github)

This repository contains global settings and templates for the **MinePrime-Ru** organization.

## Contents

- `CONTRIBUTING.md` — General contribution rules.
- `CODE_OF_CONDUCT.md` — Code of Conduct.
- `.github/PULL_REQUEST_TEMPLATE.md` — Default Pull Request template.
- `.github/CODEOWNERS` — Default code ownership rules.

## How it works

GitHub automatically uses files from this repository for all other repositories in the organization if they do not have their own similar files.

Project legal documents (Agreement, Acceptance Template) are located in the main governance repository: [mineprime-governance](https://github.com/MinePrime-Ru/mineprime-governance).
